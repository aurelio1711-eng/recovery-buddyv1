## 4. 3D and Animation Technologies

**Section Purpose:** Documents all supported technologies for 3D, animations, and interactions.

### Core Technologies

- **WebGL**: 3D rendering in browser
- **Canvas API**: 2D graphics and effects

### 3D Rendering Libraries

- **Three.js**: Most popular WebGL library
- **Babylon.js**: Microsoft-backed 3D engine
- **Spline**: Embedded 3D scenes (via runtime viewer)
- **PlayCanvas**: WebGL game engine

### Animation Libraries

- **GSAP (GreenSock)**: Industry-standard animations
- **Anime.js**: Lightweight animation library
- **Motion One**: Modern animation API
- **Framer Motion** (React): When React is used

### Scroll and Interaction Libraries

- **Lenis**: Modern smooth scrolling
- **Locomotive Scroll**: Smooth scroll with parallax
- **ScrollReveal**: Scroll-triggered animations
- **Barba.js**: Seamless page transitions

### UI and Styling Frameworks

- **Tailwind CSS**: Utility-first CSS framework
- **ShadCN UI**: Modern component library
- **DaisyUI**: Tailwind-based component kit
- **Bootstrap**: Optional fallback

### Special Effects Libraries

- **Particles.js**: Particle background effects
- **Vanta.js**: Animated 3D backgrounds
- **Hover.css**: Hover animation collection
- **Tilt.js**: 3D hover tilt effects
- **Spline Viewer**: Embed Spline scenes

This comprehensive tech stack enables cutting-edge frontend experiences with the best free tools available.
